Deskripsi Mata Kuliah Cloud Computing dengan Penggunaan GitHub dan Docker

Mata kuliah ini dirancang untuk memberikan pemahaman mendalam tentang konsep dan praktik Cloud Computing, sambil memanfaatkan alat-alat modern seperti GitHub dan Docker. Selama satu semester, siswa akan dijejali dengan pengetahuan tentang komputasi awan, infrastruktur sebagai layanan (IaaS), platform sebagai layanan (PaaS), dan perangkat lunak sebagai layanan (SaaS).

Materi Kuliah:

  1. Pengenalan Cloud Computing:
    • Konsep dasar dan model layanan Cloud Computing.
    • Keuntungan dan tantangan menggunakan teknologi awan.
  2. GitHub untuk Kolaborasi:
    • Pengenalan ke Git dan GitHub sebagai sistem kontrol versi distribusi.
    • Kolaborasi tim menggunakan fitur GitHub seperti _pull requests_, _branches_, dan _issues_.
    • Manajemen proyek dan alur kerja pengembangan yang efektif.
  3. Penggunaan Docker untuk Kontainerisasi:
    • Konsep dasar kontainer dan perbedaannya dengan virtualisasi tradisional.
    • Implementasi Docker sebagai platform kontainer yang dapat diandalkan.
    • Membangun, menyimpan, dan mendistribusikan aplikasi dalam kontainer.
  4. Orkestrasi Kontainer dengan Docker Compose dan Kubernetes:
    • Penjelasan tentang orkestrasi kontainer untuk manajemen dan penyebaran aplikasi.
    • Docker Compose untuk pengaturan dan pengelolaan aplikasi dalam lingkungan kontainer.
    • Pengenalan ke Kubernetes sebagai alat orkestrasi yang canggih.
  5. Proyek Akhir:
    • Siswa akan diberikan proyek akhir berbasis cloud menggunakan GitHub untuk kolaborasi dan Docker untuk kontainerisasi.
    • Menerapkan konsep yang dipelajari sepanjang semester untuk merancang, mengimplementasi, dan memelihara aplikasi di lingkungan awan.

Tujuan Pembelajaran:

  • Memahami prinsip-prinsip dasar Cloud Computing.
  • Menguasai alat kolaborasi dan manajemen proyek dengan GitHub.
  • Mampu menggunakan Docker untuk mengontainerisasi aplikasi.
  • Memahami konsep orkestrasi kontainer dan dapat mengimplementasikannya dengan Docker Compose dan Kubernetes.
  • Menerapkan pengetahuan dalam proyek akhir yang relevan dengan dunia industri.

Mata kuliah ini memberikan landasan yang kuat untuk mahasiswa yang ingin mengembangkan keterampilan di bidang Cloud Computing dan teknologi terkait. Dengan fokus pada praktik terbaik industri dan alat-alat modern, siswa diharapkan dapat menghadapi tantangan dunia nyata di masa depan.
